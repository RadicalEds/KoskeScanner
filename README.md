# KoskeScanner
Scans Known Filetypes for Appended Data   

Inspired By Koske Miner AI Malware   
Upon Inspection, Malware Is Appended to the End of Image Files

This Tool Detects Them.

### Currently Known Filetypes
- png
- jpg/jpeg
